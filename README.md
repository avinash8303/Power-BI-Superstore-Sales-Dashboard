# Super Store Sales Dashboard
 Interactive Sales Dashboard Built During Using PowerBI

## Overview
 This project is an interactive Power BI dashboard built to analyze Super Store sales performance.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Features
- Sales, Profit, Quantity & Average Delivery KPIs
- Regional Sales Analysis
- Category & Sub-Category Analysis
- Payment Mode & Ship Mode Analysis
- Interactive Filters (Slicers)

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- DAX
- Data Visualization
- Business Intelligence
